# 📚 StudyDash

A modern productivity dashboard built with HTML, CSS, and JavaScript.

StudyDash combines:

* Notes management
* Task tracking
* Pomodoro timer
* Study analytics
* Ambient study music
* Theme switching

into a single clean productivity workspace.

---

## 🚀 Live Demo

[Open StudyDash](https://koyalmaity007.github.io/Notes-App/Notes.html)

---

## ✨ Features

### 📒 Notes System

* Create study notes
* Organize notes by subject
* Save notes using localStorage
* Recent note preview

### ✅ Task Manager

* Add tasks with deadlines
* Priority selection
* Track completed tasks
* Dashboard statistics

### ⏳ Pomodoro Timer

* Custom focus/break durations
* Session tracking
* Productivity streak counter
* Circular progress animation

### 🎵 Ambient Study Music

* Lo-fi
* Rain
* Forest
* Cafe ambience

### 📊 Analytics Dashboard

* Weekly analytics section
* Progress indicators
* Productivity overview

### 🌙 Theme Support

* Dark mode UI
* Interactive dashboard layout
* Responsive design
